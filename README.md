# RELAT-RIO-DE-IMPLEMENTA-O-DE-SERVI-OS-AWS
RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS
Data: 15/01/2026

 Empresa: Abstergo Industries

 Responsável: Mayara Campese

Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries, realizado por Mayara Campese.

 O objetivo do projeto foi selecionar e implementar 3 serviços da AWS com foco principal na redução imediata de custos operacionais, sem comprometer o desempenho e a segurança dos sistemas da empresa.

Descrição do Projeto
O projeto de implementação foi dividido em 3 etapas, cada uma voltada para um aspecto específico da otimização de custos na infraestrutura em nuvem.



Etapa 1: AWS Cost Explorer
Foco da ferramenta: Monitoramento e análise de custos
Descrição do caso de uso:
 O AWS Cost Explorer foi implementado para permitir a visualização detalhada dos gastos da infraestrutura AWS da Abstergo Industries. A ferramenta possibilita identificar serviços com maior consumo financeiro, padrões de uso excessivo e oportunidades de economia.
 Com essa análise, a empresa conseguiu detectar recursos subutilizados e ajustar o consumo, promovendo uma redução imediata nos custos mensais.
Etapa 2: AWS EC2 Auto Scaling
Foco da ferramenta: Otimização do uso de recursos computacionais
Descrição do caso de uso:
 O serviço AWS EC2 Auto Scaling foi configurado para ajustar automaticamente a quantidade de instâncias EC2 de acordo com a demanda real do sistema.
 Dessa forma, a empresa evita custos desnecessários com servidores ociosos em períodos de baixa utilização, pagando apenas pelos recursos realmente utilizados, o que gera economia imediata e melhora a eficiência operacional.
Etapa 3: AWS S3 Intelligent-Tiering
Foco da ferramenta: Redução de custos com armazenamento
Descrição do caso de uso:
 O AWS S3 Intelligent-Tiering foi adotado para gerenciar automaticamente o armazenamento de dados, movendo arquivos entre camadas de acesso frequente e infrequente conforme o padrão de uso.
 Essa automação elimina a necessidade de gerenciamento manual e reduz custos com armazenamento de dados pouco acessados, mantendo o desempenho e a disponibilidade das informações.
Conclusão
A implementação das ferramentas na empresa Abstergo Industries tem como resultado esperado a redução significativa de custos operacionais, melhor controle financeiro sobre os recursos em nuvem e maior eficiência no uso da infraestrutura AWS.

 Recomenda-se a continuidade da utilização das ferramentas implementadas, bem como a avaliação constante de novos serviços e práticas da AWS que possam contribuir para a otimização contínua dos processos e despesas da empresa.
